# MPU6050实验(库函数版) ：STM32_MPU6050(F103+F407)

## 简介
本仓库提供了一个基于库函数的MPU6050实验代码，适用于STM32F103和STM32F407系列微控制器。通过本实验，您可以学习如何在STM32平台上使用MPU6050传感器，获取加速度和角速度数据。

## 资源内容
- **MPU6050库函数代码**：包含完整的MPU6050初始化、数据读取和处理代码。
- **STM32F103工程文件**：适用于STM32F103系列微控制器的Keil工程文件。
- **STM32F407工程文件**：适用于STM32F407系列微控制器的Keil工程文件。
- **原理图和接线图**：提供了MPU6050与STM32的接线图，方便用户进行硬件连接。

## 使用说明
1. **硬件连接**：
   - 根据提供的接线图，将MPU6050传感器正确连接到STM32开发板上。
   - 确保电源和地线连接正确，避免损坏硬件。

2. **软件配置**：
   - 打开对应的Keil工程文件（STM32F103或STM32F407）。
   - 根据您的开发板配置，修改工程中的相关参数（如I2C引脚配置）。
   - 编译并下载代码到STM32开发板。

3. **运行实验**：
   - 下载完成后，复位开发板。
   - 通过串口调试工具（如SecureCRT、Putty等）查看MPU6050的加速度和角速度数据。

## 注意事项
- 确保MPU6050传感器与STM32的I2C通信正常，避免通信错误导致数据读取失败。
- 在调试过程中，建议使用示波器或逻辑分析仪检查I2C信号，确保通信稳定。
- 如果遇到问题，请参考STM32和MPU6050的官方文档，或在本仓库的Issues中提出问题。
